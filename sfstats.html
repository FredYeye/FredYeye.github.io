Shining Force 1 stat graph

2023-09-09

This is a tool to see the target, minimum and maximum stat values per level for Shining Force 1 characters. The target stat is the stat a character "should have" for a given level; they would always have this value if there was no randomness involved.

Choose a character, stat and promotion status. The target stat value per level is shown as a line. The shaded area represents the min-max per level. Lines can be hidden by clicking the associated name above the bar. At present there is no way to delete a line without refreshing.
